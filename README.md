# wgpucube.github.io

This repo holds the contents and GitHub Pages deployment workflow for the wgpucube demo at https://wgpucube.github.io/

The wasm and JavaScript are built and deployed from the main wgpucube repository at https://github.com/tomstokes/wgpucube

The contents of this repo are mostly uninteresting if you're looking for details about the wgpucube demo. Please see the [main repository](https://github.com/tomstokes/wgpucube) for the source code of the cube demo.
